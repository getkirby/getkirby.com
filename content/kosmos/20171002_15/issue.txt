## Fresh Kirby Plugins
- [Kirby Relationship](https://github.com/olach/kirby-relationship) is a kind of sortable multi-select field. It allows you to select multiple items from a list and sort them manually. Created by Ola Christensson (free).

- Lukas Kleinschmidt's [Kirby File types plugin](https://github.com/lukaskleinschmidt/kirby-file-types) allows you to limit file selection to certain types, like images or documents (free).

- The [Kirby Anchor Headings](https://github.com/wottpal/kirby-anchor-headings) plugin by Dennis Kerzig provides a field method that enumerates heading-elements, generates IDs for anchor-links and inserts custom markup based on your options (free, pre-release).

- Jon Gacnik built the [Kirby Index Field](https://github.com/jongacnik/kirby-index-field) that shows pages in the Panel as a sortable and filterable datatable. Ideal for working with larger page sets (free). Works well with the [Hidebar field](https://github.com/jongacnik/kirby-hidebar-field) by the same developer.

-
- The [Ecwid shopping cart](https://github.com/Ecwid/kirby-ecwid-online-store) plugin lets you to add an online store and checkout to your website. Requires an [Ecwid account](https://www.ecwid.com) (plugin free).

## Kirby Tutorials
- Lauri Liimatta shares a tutorial on [how to create a comment system](http://lauriliimatta.com/blog/how-to-create-comments-system-for-kirbycms) for Kirby from scratch. If you prefer a ready to go solution, check out [this plugin](https://github.com/Addpixel/KirbyComments).

## Tools
- [Send](https://send.firefox.com/) is a Firefox Test Pilot that allows you to send files up to 1 GB to another person. In private, encrypted, self-destroying.

## Books
- [Animation at Work](https://abookapart.com/products/animation-at-work) is a new book by Rachel Nabors in the [A Book Apart](https://abookapart.com) series. It explores how to use animation to you to ease cognitive load and to design better user experiences.

## Tutorials
- For beginners: Brian Haferkamp shows [how to build a basic filtering system](https://codepen.io/brianhaferkamp/post/create-a-basic-filtering-system) for portfolio items or similar categorized content.

- When trying to submit a form on weak internet connections, users easily lose all their input, resulting in a bad user experience. We can work around this problem by following the [tips in this offline forms tutorial](https://mxb.at/blog/offline-forms/).
