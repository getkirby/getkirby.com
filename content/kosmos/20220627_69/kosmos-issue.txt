## And from the Kosmos

### Performance

Check how much JS code you could remove by using modern JavaScript with [EStimator](https://estimator.dev/).

[Lazy-load YouTube videos](https://gomakethings.com/how-to-lazy-load-youtube-videos-with-vanilla-javascript/) with vanilla JavaScript to prevent unnecessary loading of JavaScript files if the video is never played.


### PHP

[Assetic](https://github.com/assetic-php/assetic) is a PHP asset manager which included filters for image optimization, minifying & uglifying files, or parsing SASS into CSS.

[churn-php](https://github.com/bmitch/churn-php) is a package that helps you identify PHP files in your project that could be good candidates for refactoring.


### Tools & libraries

[The Knightlab](https://knightlab.northwestern.edu/) develops prototypes,projects, and services that help make information meaningful. [Projects](https://knightlab.northwestern.edu/projects/) include libraries for timelines storylines or storymaps and a few more.

Write newsletters in Markdown and send to many recipients from your machine using AWS SES with [Newsletter CLI](https://github.com/JustComments/newsletter-cli) .

[Arctype](https://arctype.com)  is an SQL client and database management tool available for macOS, Windows, and Linux. App pricing is subscription based with a free tier, account is optional, though.

[JSON Mask](https://github.com/nemtsov/json-mask) is a language and engine for selecting specific parts of a JS object, hiding the rest, while preserving the structure of the original input object.

[DevDash](https://thedevdash.com/) is a highly configurable terminal dashboard for developers and creators.
