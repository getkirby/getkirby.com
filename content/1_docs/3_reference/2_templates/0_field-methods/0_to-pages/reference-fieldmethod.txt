## Example

### Text file

```kirbycontent
  Title: My page
  ----
  Text: Some text
  ----
  Related:

  - blog/article-a
  - blog/article-b
```

### Template

```php
<h2>Related articles</h2>
<ul>
  <?php foreach ($page->related()->toPages() as $related): ?>
  <li>
    <a href="<?= $related->url() ?>">
      <?= $related->title() ?>
    </a>
  </li>
  <?php endforeach ?>
</ul>
```
