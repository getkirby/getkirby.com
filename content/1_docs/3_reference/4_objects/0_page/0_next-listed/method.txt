## Example

```php
<?php if($next = $page->nextListed()): ?>
<a href="<?= $next->url() ?>">next listed page</a>
<?php endif ?>
```

### Pass a collection

```php
<?php
$collection = $page->siblings()->listed()->sortBy('date', 'desc');
if($next = $page->nextListed($collection)): ?>
<a href="<?= $next->url() ?>">next listed page</a>
<?php endif ?>
```
