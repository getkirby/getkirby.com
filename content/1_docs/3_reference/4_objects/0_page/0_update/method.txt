Auth: pages.update

----

Text:

## Example

```php
try {

  page('mypage')->update([
    'title'        => 'A new title',
    'text'         => 'Some text',
    'anotherfield' => 'Some more data'
  ]);

  echo 'The page has been updated';

} catch(Exception $e) {

  echo $e->getMessage();

}

```

### Updating a field by callback

```php
$page->update([
  'title' => function($title) {
    return Str::lower($title);
  }
]);
```

## Multi-language site
Use the optional `$lang` parameter to specify the language you want to update.

```php
try {

  page('mypage')->update([
    'title'        => 'A new title',
    'text'         => 'Some text',
    'anotherfield' => 'Some more data'
  ], 'en');

  echo 'The page has been updated';

} catch(Exception $e) {

  echo $e->getMessage();

}

```
