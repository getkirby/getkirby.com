## Example

```php
<?php if($next = $page->next()): ?>
<a href="<?= $next->url() ?>">next page</a>
<?php endif ?>
```

### Pass a collection

```php
<?php
$collection = $page->siblings()->listed()->sortBy('date', 'desc');
if($next = $page->next($collection): ?>
<a href="<?= $next->url() ?>">next page</a>
<?php endif ?>
```
