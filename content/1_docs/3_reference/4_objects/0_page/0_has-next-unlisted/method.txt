## Example

```php
if ($page->hasNextUnlisted()) {
  echo $page->nextUnlisted()->url();
}
```

### With collection as argument

```php
$collection = $page->siblings()->unlisted()->sortBy('date', 'desc');
if ($page->hasNextUnlisted($collection)) {
  echo $page->nextUnlisted($collection)->url();
}
```
