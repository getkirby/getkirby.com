## Example

```php
<?php if($prev = $page->prev()): ?>
<a href="<?= $prev->url() ?>">previous page</a>
<?php endif ?>
```

### With collection as argument

```php
<?php
$collection = $page->siblings()->listed()->sortBy('date', 'desc');
if($prev = $page->prev($collection)): ?>
<a href="<?= $prev->url() ?>">previous page</a>
<?php endif ?>
```
