## Example

```php
<?php if($prev = $page->prevListed()): ?>
<a href="<?= $prev->url() ?>">previous listed page</a>
<?php endif ?>
```

### With collection as argument

```php
<?php
$collection = $page->siblings()->listed()->sortBy('date', 'desc');
if($prev = $page->prevListed($collection)): ?>
<a href="<?= $prev->url() ?>">previous listed page</a>
<?php endif ?>
```
