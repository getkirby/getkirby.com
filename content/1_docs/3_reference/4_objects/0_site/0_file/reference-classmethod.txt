## Example

### Fetching the first file

```php
<?php if($file = $site->file()): ?>
<a href="<?= $file->url() ?>">
  <?= html($file->filename()) ?>
</a>
<?php endif ?>
```

### Fetching a specific file

```php
<?php if($file = $site->file('myfile.pdf')): ?>
<a href="<?= $file->url() ?>">
  <?= html($file->filename()) ?>
</a>
<?php endif ?>
```
