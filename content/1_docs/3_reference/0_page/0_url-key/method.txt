Title:

{{class.instance}}->urlKey($lang = null)

----

Excerpt:

Returns the URL key from the content file if available and otherwise returns the page UID

----

Params: -
  name: $lang
  type: string
  text:

----

Return:

type: string
text:

----

Text:

Method only exists for multi-language sites.

## Example

```php
<?= $page->urlKey() ?>

```
