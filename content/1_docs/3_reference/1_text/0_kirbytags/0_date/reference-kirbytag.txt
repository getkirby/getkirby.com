Title:

date

----

Excerpt: Adds a dynamic date to the text

----

Text:

## Examples and options:

### Current year

```kirbytext
(\date: year)
```

### Any date defined by a PHP date format string

```kirbytext
(\date: d.m.Y H:i:s)
```

You can read more about all (link: https://www.php.net/manual/en/function.date.php text: available formatting options on the PHP website).
