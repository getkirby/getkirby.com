Title: (:any)

----

Excerpt: Matches any character until the next slash

----

Text:

## Example

```php "/site/config/config.php"
return [
  'routes' => [
    [
      'pattern' => '/projects/(:any)',
      'action'  => function (string $path) {
        // react to requests
      }
    ]
  ]
];
```
