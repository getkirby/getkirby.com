Title:

$files->pluck($field, $split = null, $unique = false)

----

Excerpt:

Plucks all values for a specific field from the collection and returns those as array.

----

Return:

type: array
text:

----

Text:

## Example

```php
<ul>
  <?php foreach($page->files()->pluck('filename') as $filename): ?>
  <li><?= html($filename) ?></li>
  <?php endforeach ?>
</ul>
```
