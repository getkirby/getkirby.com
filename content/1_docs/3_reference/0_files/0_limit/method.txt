Title:

$files->limit($limit)

----

Excerpt:

Returns the collection with a limited set of items

----

Return:

type: $files
text:

----

Text:

## Example

```php
<ul>
  <?php foreach($page->files()->limit(3) as $file): ?>
  <li>
    <a href="<?= $file->url() ?>">
      <?= html($file->filename()) ?>
    </a>
  </li>
  <?php endforeach ?>
</ul>
```
