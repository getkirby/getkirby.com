Title:

$files->slice($offset = null, $limit = null)

----

Excerpt:

Returns a sliced version of the collection starting from a given offset and an optional limited number of items.

----

Return:

type: $files
text:

----

Text:

## Example

```php
<ul>
  <?php foreach($page->files()->slice(5,10) as $file): ?>
  <li>
    <a href="<?= $file->url() ?>">
      <?= html($file->filename()) ?>
    </a>
  </li>
  <?php endforeach ?>
</ul>
```
