Title:

$files->keys()

----

Excerpt:

Returns an array with all filenames in the collection.

----

Return:

type: array
text:

----

Text:

## Example

### Example Files

- document.pdf
- numbers.xls
- myimage.jpg
- download.zip

```php
<?php dump($page->files()->keys()) ?>
/*
[
  'document.pdf',
  'numbers.xls',
  'myimage.jpg',
  'download.zip'
]
*/
```
