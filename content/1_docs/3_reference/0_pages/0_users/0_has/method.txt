Title:

{{class.instance}}->has($user)

----

Excerpt:

Checks if a $user object is included in the collection

----
Params: -
  name: $user
  type: User | string
  text: Accepts a username or a user object

----

Return:

type: boolean
text:

----

Text:

```php
<?php if($kirby->users()->has('someone')): ?>
There exists a user with the username `someone`
<?php endif ?>
```
