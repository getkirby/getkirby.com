Title: Navigation option
----
Text:

```yaml
# /site/blueprints/pages/default.yml

title: Simple Page

# with all
navigation:
    status: all
    template: all

# with specific selects
navigation:
    status:
       - draft
       - unlisted
    template:
       - album
       - default

# with single select and custom sort
navigation:
    status: listed
    template: default
    sortBy: title asc
```
