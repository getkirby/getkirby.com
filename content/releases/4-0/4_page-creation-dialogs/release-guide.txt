----

Subtitle: Faster, better page creation

Intro: You can customize the create dialog for a new page by adding fields, changing the label or the title field and disable the automatic redirect to the new page.

Customize the page creation dialog to your needs.

Give the title a fitting label, add new fields or disable redirecting to the new page.

Text:

## Configuration

```yaml
title: Product
create:
  title:
    label: Product name
  fields:
    - price
    - brand
  redirect: false
```

(image: page-creation-dialog.png)

<info>
The list of allowed fields inside the dialog is limited to field types that don't open their own dialog. This excludes the following field types: blocks, color, files, layout, link, object, pages, structure,users, as well as custom fields.
</info>

## Custom Page Creation Status

You can now also define a status for your newly created pages.

```yaml
create:
  status: listed
```
