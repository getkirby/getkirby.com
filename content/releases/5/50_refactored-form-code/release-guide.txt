Title: Refactored Forms

----

Text:

Weâ€™ve refactored our Form and Field classes to provide a more stable and usable Form API for our models. You can now use the same Form API for your own projects to create forms in your templates.

**See the `Kirby\Form\Form` documentation for more information.**

```php
use Kirby\Form\Form;

$form = new Form(
    fields: [
        'name' => [
            'label' => 'Name',
            'type'  => 'text'
        ],
        'email' => [
            'label' => 'Email',
            'type'  => 'email'
        ],
        'message' => [
            'label' => 'Message',
            'type'  => 'textarea'
        ]
    ]
);

$form->fill([
    'name'  => 'Peter Griffin',
    'email' => 'mail@example.com'
]);

$form->toFormValue();
```
