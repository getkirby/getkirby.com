Title: Kirby Inherit

----

Repository: https://github.com/youngcut/kirby-inherit

----

Category: templating

----

Subcategory: field-method

----

Description: If the current field is empty or does not exist, it gets the same field from the parents pages. It goes all the way to the site.

----

Example:

```php
<?= $page->myfield()->inherit() ?>
```
