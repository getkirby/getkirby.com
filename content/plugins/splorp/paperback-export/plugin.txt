Title: Paperback Export

----

Repository: https://github.com/splorp/kirby-paperback-export

----

Category: dev-tools

----

Subcategory: utilities

----

Description: Generates a lightly formatted text file based on your site content, which can be used to create a “book” viewable on a Newton OS device.

----

Example:

```
yoursite.com/export/paperback
```
