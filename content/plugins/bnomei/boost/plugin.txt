Title: Boost

----

Repository: https://github.com/bnomei/kirby3-boost

----

Category: performance

----

Subcategory: caching

----

Description: Boost the speed of Kirby by having content files of pages cached, with automatic unique ID, fast lookup and Tiny-URL.

----

Example:

```php
// 1) set a cache driver in site/config/config.php
return [
    'bnomei.boost.cache' => [
        'type' => 'apcu', // php, redis, mysql, sqlite ...
    ],
];

// 2) add boostid field in site/blueprints/pages/blogpost.yml

// 3) extend model in site/models/blogpost.php
class BlogpostPage extends \Kirby\Cms\Page
{
    use \Bnomei\PageHasBoost;
}
```
