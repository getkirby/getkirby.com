Title: Highlight

----

Repository: https://github.com/S1SYPHOS/kirby3-highlight

----

Category: templating

----

Subcategory: syntax-highlighting

----

Description: Themeable server-side syntax highlighting

----

Example:

```php
return [
    's1syphos.highlight' => [
        'class'     => 'hljs',
        'languages' => ['html', 'php'],
        'escaping'  => false
    ]
];
```
