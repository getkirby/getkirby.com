Title: Recurr

----

Text:

----

Repository: https://github.com/HashandSalt/kirby-recurr

----

Category: dates

----

Description: This plugin integrates the Recurr library to make it easier to work with repeating dates (like events on a calender).

----

Example:

```php
$datelist = $site->recurr(
    '2022-09-10 20:00:00',
    '2022-09-11 02:00:00',
    'WEEKLY',
    ['WE', 'TH', 'FR'],
    '2022-10-11'
);
```

----

Versions: 3,4
