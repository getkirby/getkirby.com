Title: Options

----

Excerpt:

Kirby comes with sensible default settings, but you can use its config options to change Kirby's behavior and functionality according to your preferences. You can even set environment specific options by providing additonal, domain-specific config files.

In addition to the config options listed below you can define your own config settings for use in plugins, templates etc.

----

Config options can be set in your config file or in a plugin.
