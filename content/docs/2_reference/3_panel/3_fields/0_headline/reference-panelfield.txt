Title: Headline

----

Intro:

Creates a headline to group fields

----

Text:

The headline field helps to group larger sets of fields. It creates a headline with additional white-space and a prepended index number.

(image: headline.png)

## Example

```yaml
fields:
  pageSettings:
    label: Page Settings
    type: headline
  type:
    label: Page type
    type: text
  state:
    label: Draft
    type: toggle
  content:
    label: Content
    type: headline
  text:
    label: Text
    type: textarea
```
