## Example

### Fetching the first image

```php
<?php if($image = $site->image()): ?>
<img src="<?= $image->url() ?>" alt="">
<?php endif ?>
```

### Fetching a specific image

```php
<?php if($image = $site->image('myimage.jpg')): ?>
<img src="<?= $image->url() ?>" alt="">
<?php endif ?>
```
