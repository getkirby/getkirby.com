## What is an image file?

Kirby considers the following file types as images:

- jpeg
- jpg
- jpe
- gif
- png
- svg
- ico
- tif
- tiff
- bmp
- psd
- ai
- eps
- ps

----

Examples:

```php
<?php
echo $page->hasImages(); // will echo 1 if true, nothing if false

if ($page->hasImages()) {
  $images = $page->images();
}
?>
```
