## What is an audio file?

Kirby considers the following file types as audio files:

- mp3
- m4a
- wav
- aif
- aiff
- midi

----

Examples:

```php
<h2>Songs</h2>

<?php foreach($page->audio() as $audio): ?>
<audio controls>
  <source src="<?= $audio->url() ?>" type="<?= $audio->mime() ?>">
</audio>
<?php endforeach ?>
```
