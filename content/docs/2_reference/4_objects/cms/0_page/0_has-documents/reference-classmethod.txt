## What is a document file?

Kirby considers the following file types as document:

- text
- mdown
- md
- markdown
- pdf
- doc
- docx
- dotx
- word
- xl
- xls
- xlsx
- xltx
- ppt
- pptx
- csv
- rtf
- rtx

----

Examples:

```php
echo $page->hasDocuments(); //returns the number of documents in the folder

if($page->hasDocuments()) {
  $documents = $page->documents();
}
```
