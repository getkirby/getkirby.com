## What is a video file?

Kirby considers the following file types as video:

- mov
- movie
- avi
- ogg
- ogv
- webm
- flv
- swf
- mp4
- m4v
- mpg
- mpe

----

Examples:

```php
<?php
echo $page->hasVideos(); // will echo 1 if true, nothing if false

if($page->hasImages()) {
  $videos = $page->videos();
}
?>
```
