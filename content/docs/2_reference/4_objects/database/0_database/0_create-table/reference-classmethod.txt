Examples:

```php
$database->createTable('product', [
    'id'          => ['type' => 'id'],
    'code'        => ['type' => 'varchar', 'size' => 50],
    'description' => ['type' => 'varchar'],
    'quantity'    => ['type' => 'int', 'unsigned' => false],
    'price'       => ['type' => 'decimal', 'precision' => 12, 'decimal_places' => 4]
]);
```
