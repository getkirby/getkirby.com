## Example

```php
<?php if($next = $page->nextUnlisted()): ?>
<a href="<?= $next->url() ?>">next unlisted page</a>
<?php endif ?>
```

### Pass a collection

```php
<?php
$collection = $page->siblings()->unlisted()->sortBy('date', 'desc');
if($next = $page->nextUnlisted($collection)): ?>
<a href="<?= $next->url() ?>">next unlisted page</a>
<?php endif ?>
```
