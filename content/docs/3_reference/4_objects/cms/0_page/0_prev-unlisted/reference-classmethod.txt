## Example

```php
<?php if($prev = $page->prevUnlisted()): ?>
<a href="<?= $prev->url() ?>">previous unlisted page</a>
<?php endif ?>
```

### With collection as argument

```php
<?php
$collection = $page->siblings()->unlisted()->sortBy('date', 'desc');
if($prev = $page->prevUnlisted($collection)): ?>
<a href="<?= $prev->url() ?>">previous unlisted page</a>
<?php endif ?>
```
