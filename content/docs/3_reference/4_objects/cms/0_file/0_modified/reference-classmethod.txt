## Example

```php

if($file = $page->file()) {
    // unix timestamp
    echo $file->modified();

    // formatted date
    echo $file->modified('d/m/Y');

    // formatted date using strftime
    echo $file->modified('%d/%m/%Y', 'strftime');

}
```

<info>
The modification time reflects the actual file as well as the `.txt` content file(s), whichever was last modified.
</info>
