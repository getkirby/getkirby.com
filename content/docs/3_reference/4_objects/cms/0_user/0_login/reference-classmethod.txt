## Example

```php
// simple login form submission example
if($username = get('username') and $password = get('password')) {

  $user = $kirby->user($username);

  if($user and $user->login($password)) {
    // redirect to the homepage
    // or any other page
    go();
  } else {
    echo 'invalid username or password';
  }

}
```
