<alert until="3.7.0">
The `pages()` helper throws a deprecation warning if a single ID gets passed. Starting in 3.7.0 it will awlays return a `Kirby\Cms\Pages` collection. Use (link: docs/reference/templates/helpers/page text: `page()`) for a single page instead.
</alert>

## Example

```php
<?php $collection = pages(['home', 'blog']) ?>
<?php $collection = pages([$page, $page->children()->first()]) ?>
```
<info>
The `pages()` helper fetches published pages only; ids of draft pages are ignored.
</info>
